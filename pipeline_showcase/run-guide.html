Run Guide

How to start the project from the repo.

This frontend can guide you through the project, but the Python app itself must be started from the terminal.

Commands

Use these commands in the project root.

pip install -r requirements.txt
python main.py

The launcher in main.py calls the gesture pipeline and opens the webcam preview window.

Shutdown

Close the camera completely with one action.

Press Q, press Esc, or click the preview window close button once. The app releases the camera and destroys the OpenCV windows.

Why not run from HTML?

A static frontend cannot execute local Python directly.

Browsers do not allow a normal local HTML page to launch Python code on your machine automatically. That is why this page gives you the exact command instead of pretending the button can start the app.
