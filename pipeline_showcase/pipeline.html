Pipeline Overview

From webcam frame to mouse action.

This page explains the complete virtual-mouse pipeline in the order it happens inside the Python application.

01

Frame Capture

OpenCV opens the webcam and reads a frame continuously so the program always has fresh visual input.

02

Preprocessing

The frame is flipped horizontally to feel natural and converted from BGR to RGB so it can be passed into MediaPipe correctly.

03

Hand Landmarker

MediaPipe Tasks runs the hand-landmarker model and returns 21 hand landmarks for the detected hand.

04

Gesture Logic

The code checks key fingertip distances and positions to decide if the user means move, left click, right click, or scroll.

05

OS Control

PyAutoGUI converts the interpreted gesture into real operating system actions such as moving the cursor or clicking.

06

Visual Feedback And Exit

OpenCV draws status text and landmarks on screen, and the app closes fully when the user presses Q, Esc, or the window close button once.
